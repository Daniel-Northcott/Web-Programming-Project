About

This website catalogs current movies with ratings and reviews.

What is Movie Addicts?

Movie Addicts is a full-stack web application for movie lovers to share their opinions, discover films, and view ratings from others. The website includes a homepage highlighting currently trending and admin endorsed films, a movie catalog with user-submitted reviews, and user authentication features.

Users can create accounts, and submit reviews when logged in, while unregistered users will be restricted to browse and read-only.

Project Modules
  • Users: Login, registration, sessions, and profile handling.
  • Movies: A database of current movies with posters and details.
  • Ratings: Allows users to rate films from 1 to 5 stars.
  • Reviews Module: users can write, read, and interact with movie reviews.
Technical Details
  • Frontend: HTML, CSS, and JavaScript for responsive UI and dynamic interaction.
  • Backend: Node.js with Express to handle routing and server logic.
  • Database: MongoDB for storing users, movies, ratings, and reviews.
  • Authentication: Session-based login using localStorage for user state.
  • API Integration: External movie API for fetching movie data and posters.
Functionalities
  • User registration and login with session persistence.
  • Browse and search a catalog of current movies.
  • View movie details including ratings and reviews.
  • Submit ratings and written reviews.
  • Read and interact with reviews from other users.
  • Logout functionality and dynamic navigation updates.
